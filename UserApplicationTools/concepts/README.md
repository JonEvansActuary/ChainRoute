# Concepts Index

Concept documents for each user-facing app or cloud service. See the [main README](../README.md) for ecosystem overview and priorities.

| # | Document | Platform | Primary job |
|---|----------|----------|-------------|
| 01 | [Web Verifier](./01-web-verifier.md) | Web | Verify chain from link, tx hash, or QR; show timeline and status. |
| 02 | [Web Builder (Provenance Studio)](./02-web-builder-studio.md) | Web | Wizard: genesis → events (supports → blob → sign anchor). |
| 03 | [Mobile Apps](./03-mobile-apps.md) | Smartphone | Capture evidence; verify at point of sale (scan QR). |
| 04 | [Desktop App](./04-desktop-app.md) | Desktop | Full build with Ledger, key store, batch sign. |
| 05 | [Verification API](./05-verification-api.md) | Cloud API | Verify chain by tx/genesis; return structured result for any client. |
| 06 | [Provenance API / PaaS](./06-provenance-api-paas.md) | Cloud API | Orchestrate or fully run build (optional custody). |
| 07 | [AI Assistants and Agents](./07-ai-assistants-agents.md) | Any | Build: suggest events/summaries; Verify: explain chain, Q&A. |
| 08 | [Browser Extension](./08-browser-extension.md) | Browser | “Verify” badge on pages with ChainRoute links. |

All concepts align with the [ChainRoute Protocol](../../protocol.md) and existing [docs/code](../../docs/code).
